# OrdenacaoDeValoresEmJava
Este programa foi desenvolvido em Java com o objetivo de praticar estruturas de decisão.
A aplicação recebe três valores inteiros e uma opção fornecida pelo usuário, permitindo exibir os valores em diferentes formatos: Ordem crescente; Ordem decrescente; Maior valor posicionado no meio.

Para a resolução do problema, foi utilizada lógica condicional (if/else) para identificar o maior, o menor e o valor intermediário, além da estrutura switch-case para selecionar a forma de exibição dos dados.
Projeto desenvolvido como exercício de lógica de programação.
